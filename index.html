AI Playerbot by ike3

This is a fork of playerbot with completely different AI based on strategies.

Welcome to the AI Playerbot project! This is a modification of MaNGOS server which allows you to use your guild characters as bots. This includes:

  • A set of actions that bot can do (i.e., cast a frost nova)
  • A set of triggers telling bot that something is happened (i.e., enemy is too close)
  • A set of strategies that combines actions and triggers making bot to react to world events.
  • Structured and maintainable source code.
The main goal of this project is learning and having fun.

This makes it possible the use cases below:

  • Inviting bots to party and playing dungeon or raid
  • Inviting bots to party and doing some team PVP
  • Summoning your alts for bots to quick item echange, casting some spells, buffing, etc.
  • Random PVP with bots of opposite faction
  • Dueling
  • Making World of Warcraft even completely single-player game!
